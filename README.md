## 🚀 Quick Start

1. **Prerequisites**: .NET 10 SDK, Node.js.
2. **Run Server**: `cd src/Services/Winnow.API && dotnet run`
3. **Run Client**: `cd src/Apps/Winnow.Client && npm install && npm run dev`
