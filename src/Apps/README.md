# Winnow Application Layer

The application layer contains the user-facing interfaces for the Winnow platform.

## 📂 Applications Overview

- [**Winnow.Client**](./Winnow.Client/README.md): The primary observability dashboard for developers (React/Vite).
- [**Winnow.Marketing**](./Winnow.Marketing/README.md): The public-facing product site and onboarding flow (React/Vite).

## 🏗 Shared Architecture

- **React & Vite**: Both applications use the modern Vite build tool for optimized development and production builds.
- **Tailwind CSS**: A unified design system is enforced through shared Tailwind configurations.
- **Component Driven**: We follow atomic design principles to ensure consistent UI across both apps.
